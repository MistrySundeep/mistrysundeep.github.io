Timesheet app

This is my first complete app in using Java. This is a copy of the core functions of the current Timesheet system. Creating this tested my knowledge on collections in java. This app was something I could use to generate an idea of my hours in a week and the amount I will likely get on pay day. Before this I had an understanding on maps and lists but I needed to learn about sorted maps and sorted lists to help me acheive my ideas. There is always room for improvement and with time I would like to understand Swing to be able to include a GUI for this.

When run the app will prompt the user to enter the hours they worked for days Monday-Sunday. This will repeat 4 times to complete a month. Once entered the app will output file with a timesheet for the users entered values. It will calculate the amount the should be paid based off a set hourly rate.
